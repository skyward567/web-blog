Здесь вы найдёте заметки об изученных языках, разборы технологий и мысли о современной разработке программного обеспечения.

Специальность: "Информатика и вычислительная техника"

  • Java Backend Developer — Hyperskill.org (JetBrains)

Опыт работы:

Опыта в коммерческой разработке программного обеспечения в рамках каких-либо организаций пока нет, стажировок в компаниях не проходил.

В рамках учёбы в университете и на дополнительных интернет-курсах выполнялись небольшие проекты, как правило индивидуально. Опыт командной работы складывается из совместного выполнения лабораторных работ и небольших проектов бригадами в рамках университета. Командная работа способна ускорить и улучшить итоговый результат, однако требует ответственного подхода и профессионализма от каждого члена команды.

Навыки:

  • Языки программирования (университет): C, C++, C#, Java, JavaScript, HTML, PostgreSQL
  • Языки программирования (школа): Python (ЕГЭ по информатике)
  • Углублённо: Java — современный платформонезависимый язык, изучаю сверх учебной программы
  • Платформы и инструменты: Visual Studio Code, Git, IntelliJ IDEA, PyCharm

Подход к обучению:

В свободное время стараюсь повышать свой уровень знаний в программировании, используя интернет-курсы. Курсы выбираю по принципу современности и востребованности языков программирования, их полезности для получения работы. В частности, углублённо изучаю Java, так как считаю его современным платформонезависимым языком с широким применением в бэкенд-разработке.
